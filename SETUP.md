# Cara Memasang GitHub Profile README

Paket ini sudah disiapkan khusus untuk username GitHub: `rullkuch`.

## Langkah pemasangan

1. Masuk ke GitHub.
2. Tekan tombol **+** lalu pilih **New repository**.
3. Isi nama repository tepat seperti ini: `rullkuch`
4. Pilih **Public**.
5. Jangan centang README karena file README sudah tersedia di paket ini.
6. Tekan **Create repository**.
7. Pilih **uploading an existing file**.
8. Upload:
   - `README.md`
   - folder `assets` beserta seluruh isinya
9. Tekan **Commit changes**.
10. Buka halaman profil GitHub. Tampilan akan langsung muncul.

## Mengganti link website

Buka `README.md`, lalu cari:

https://YOUR-WEBSITE-LINK.com

Ganti dengan link website milikmu.

## Mengganti link sosial media

Cari:

https://YOUR-SOCIAL-LINK.com

Ganti dengan link Instagram, TikTok, X, atau sosial media lain.

## Menambahkan link project

Setelah repository Discord Bot atau Roblox dibuat, kamu dapat mengubah bagian Featured Projects menjadi gambar yang bisa diklik.
